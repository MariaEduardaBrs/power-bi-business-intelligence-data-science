# Dashboard Analítico de Vendas Globais

Dashboard desenvolvido no Microsoft Power BI para análise de vendas globais.

## Objetivos
- Analisar vendas globais por categoria
- Avaliar média de desconto por subcategoria
- Comparar vendas por país e prioridade
- Explorar segmentação por país, segmento e ano

## Perguntas de Negócio

O dashboard foi desenvolvido para responder às seguintes perguntas:

1. Qual o valor total vendido?
2. Quantas vendas foram realizadas por categoria de produto?
3. Quantas vendas foram realizadas por país considerando a prioridade de entrega?
4. Qual foi a média de desconto nas vendas por subcategoria de produto?
5. Quais países tiveram maior média de valor de venda?

## Filtros disponíveis
O dashboard permite análise interativa através dos filtros:
- Ano
- Segmento
- País

## Ferramentas utilizadas
- Power BI
- DAX
- Modelagem de Dados

## Indicadores analisados
- Total de vendas globais
- Média de desconto
- Vendas por categoria
- Vendas por país
- Prioridade de pedidos
