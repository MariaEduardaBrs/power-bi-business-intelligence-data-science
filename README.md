# Microsoft Power BI Para Business Intelligence e Data Science

Repositório com projetos, dashboards e estudos desenvolvidos durante o curso **Microsoft Power BI Para Business Intelligence e Data Science** da Data Science Academy.

## Conteúdos abordados
Business Intelligence
Análise de Dados
Modelagem de Dados
Relacionamentos e DAX
Power Query e Linguagem M
SQL Analytics
Estatística aplicada
Machine Learning
Inteligência Artificial
Power Platform

## Objetivo
Consolidar conhecimentos em BI e análise de dados através da criação de dashboards, tratamento de dados, modelagem e desenvolvimento de soluções analíticas com Power BI.

## Ferramentas utilizadas
- Power BI
- Power Query
- DAX
- SQL
- Excel
- Power Platform
  
## Estrutura do repositório
Cada pasta representa um módulo/projeto desenvolvido ao longo do curso, contendo:
- arquivos .pbix
- bases de dados
- imagens dos dashboards
- anotações e aprendizados
